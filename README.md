# My Portfolio

Welcome to my portfolio! This is a dynamic, interactive project, designed as part of my portfolio to showcase my abilities in the modern web development frameworks and libraries. 

## Portfolio Objectives

The main objectives of this project are:

1. **Showcase proficiency in React**: This project is built using React, a popular framework for building user interfaces. I've used React's state and effects hooks, higher order functions, and other features to manage the app's functionality.

2. **Demonstrate understanding of animations**: The app uses the `react-spring` library for animations, demonstrating my understanding of spring physics and ability to create smooth, natural motion in web interfaces.

3. **Demonstrate understanding of user interaction**: The app uses the `react-use-gesture` library to provide drag and swipe interactions, demonstrating my ability to create interactive user experiences.

4. **Showcase ability to create a responsive design**: The card stack is fully responsive, providing a consistent experience across devices of different sizes.

## Features

- Stack of cards with detailed content about different topics.
- Users can drag and swipe the top card to the left or right to animate it away and bring the next card to the top.
- The background color of the page transitions according to the top card on the stack, providing a visual cue for the currently active card.
- The card stack is responsive and works smoothly on mobile devices.

## Libraries used

- React
- react-spring
- react-use-gesture
